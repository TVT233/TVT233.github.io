Research experience

Pruning and parameter tuning of CNNs for image recognition

[code]

I parametrized a CNN for crack recognition (including parameters such as network depth, input size, etc.) with the goal of balancing energy consumption and computational effort with accuracy for subsequent deployment of such a network on energy- and arithmetic-limited edge devices.

Parallel scheduler for heterogeneous computing platforms using greedy algorithms

[code]

I have reproduced a parallel scheduler using a greedy algorithm. It is a heuristic algorithm that batches a number of upcoming tasks and uses the greedy algorithm to schedule them on different CPUs and GPUs in each batch, so that the overall energy consumption and makespan is optimised (depending on the user’s needs).
My companion later built on this algorithm to explore better results using genetic algorithms for different batches of tasks.
